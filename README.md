# Bender Bot

> This project exposes in the form of an express API the service layer with endpoints, webhooks and sockets.
> 
> Bot to link plataforms:
- Slack multiple teams
- Rocket.Chat
- Web

> Future:
- Facebook
- Twitter
- Discord
- Telegram

> Integrations
- Airtable
- Google
- Strava
- Notion

## **Prerequisitos**

- node >= 14.17.x
- yarn >= 1.22.x
- [Botkit.ai](https://botkit.ai/)


## **Install**

```sh
yarn install
```

## **Usage**

```sh
yarn start
```
